# HarvestIQ Crop Analytics Cloud

A Flask-based Crop Management and Analytics System.

## Features

- User Login
- Dashboard
- Add Crop
- View Crops
- Edit Crop
- Delete Crop
- Crop Reports
- SQLite Database
- Docker Support

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Docker

## Run Locally

```bash
python app.py
```

## Run Using Docker

```bash
docker build -t harvestiq .
docker run -p 5001:5000 harvestiq
```

Visit:

http://localhost:5001
